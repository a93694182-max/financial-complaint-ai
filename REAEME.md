# 금융 민원 분석 AI

## 프로젝트 소개

금융 민원 데이터를 분석하고
머신러닝을 활용하여
민원 유형을 자동 분류하는
웹 대시보드입니다.

---

## 기술 스택

- Python
- Pandas
- Streamlit
- SQLite
- Scikit-learn

---

## 주요 기능

### 민원 검색 및 필터

검색어와 유형별 필터 제공

### 키워드 분석

민원 키워드 TOP10 분석

### 머신러닝 기반 분류

Naive Bayes 기반 자동 분류

### 예측 확률 시각화

AI 예측 확률 제공

### 예측 기록 저장

SQLite DB 저장

### 워드클라우드

유형별 핵심 키워드 시각화
